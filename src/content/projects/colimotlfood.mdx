Desarrollé una aplicación de pedidos de comida a domicilio con pasarela de pago. Incluyó funcionalidades como catálogo de productos, carrito de compras, historial de pedidos, seguimiento en tiempo real y notificaciones push. La app fue construida con React Native y Expo, conectada a un backend Laravel con autenticación y gestión de usuarios.

React Native, Expo, PHP, Laravel, MySQL, Firebase.
