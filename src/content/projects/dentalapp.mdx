Desarrollé una plataforma web y móvil para la gestión de clínicas dentales, incluyendo pagos y notificaciones push. El sistema permitió agendar citas, administrar historiales clínicos, generar reportes y enviar recordatorios automáticos a pacientes. Integré funcionalidades tanto administrativas como operativas para mejorar la eficiencia del consultorio.

PHP, Laravel, MySQL (web) | Ionic, TypeScript, Laravel, MySQL (móvil).
