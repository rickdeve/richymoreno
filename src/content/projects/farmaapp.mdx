Desarrollé un sistema de control de inventario farmacéutico con administración de usuarios, roles y permisos. La plataforma permitió registrar entradas y salidas de productos, generar reportes de stock y vencimientos, así como controlar accesos según nivel de usuario. Se utilizó Angular para la interfaz dinámica y Laravel como backend robusto.

PHP, Laravel, MySQL, Angular.
