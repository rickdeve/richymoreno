Desarrollé un sistema de punto de venta personalizado para Pizza Happy, incluyendo gestión de pedidos, control de inventarios, promociones, administración de combos y descuentos. El sistema también permitió integración básica con plataformas de entrega y generación de reportes de ventas diarios y mensuales.

PHP, Laravel, HTML, Bootstrap, jQuery, MySQL.
